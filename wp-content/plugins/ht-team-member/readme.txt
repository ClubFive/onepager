The WP Team Member is a elementor addons, visual composer addons, WordPress Default widgets and Ready Shortcode for WordPress.

== Description ==
WP Team Member is a WordPress Team Member / Widget Plugin. Create Team Member in a preferred style and place it to your website in any section easily. 

This plugin includes Elementor Addon, WP Bakery Page Builder addon, WordPress Widget, 5 Custom styles with different customization option, Slider enable/disable option and column control option.

== Installation ==
This section describes how to install the WP Team Member Plugins for WordPress get it working.

= 1) Install =

1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "WP Team Member".
3. Install, then Activate it.

= OR: =
1. Unzip (if it is zipped) and Upload `ht-team-member` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
1. Go to the WP News Settings Options.
2. Permalink Change Options (If you want change).
